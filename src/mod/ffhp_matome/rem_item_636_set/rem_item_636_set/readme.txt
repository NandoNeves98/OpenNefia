ブログで配布している差し替え素材の改変はご自由にどうぞ。

再配布もこちらの配布元を明記した上なら可です。※作者を偽っての配布や転載はご遠慮ください。

配布している素材は「Elona用」の差し替え素材なので、Elona以外で利用したり別のゲームに使用する事はできません。

カスタムアイテムやヴァリアントへ組込む際は同梱するtxtなどに当方名などの記載をお願いします。
